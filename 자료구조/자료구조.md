# 자료구조
> 가장 기초적인 형태 : 변수, 배열, 레코드(클래스)

목적
----------
* 추상화 : 현실의 개념과 구조를 자료구조로 표현
* 최적화

**비트마스크**
----------
* 이진수 표현을 자료구조로 쓰는 것.
* 컴퓨터는 이진법 연산이 빠르기 때문에 사용한다.
```
  + 더 빠른 수행시간
  + 더 간결한 코드
  + 더 작은 메모리 사용량
  + 연관배열을 배열로 대체(ex. Map -> int[])
```
    ※ 유의할 점
      * 비트연산자 우선순위  --> 괄호 사용 (ex. (6&4)==4)
      * 64비트 오버플로  --> 1(32비트 취급)뒤에 접미사 ull사용
      * 시프트할 때  --> 부호없는 정수형을 사용
